# External Integrations

**Analysis Date:** 2026-03-21

## APIs & External Services

**AI/Language Models:**
- Anthropic Claude API - Content summarization and structured trend extraction
  - SDK/Client: `anthropic` Python package
  - Auth: `ANTHROPIC_API_KEY` environment variable
  - Used in: `newsletter-digest/summarizer.py`, `newsletter-digest/trend_extractor.py`
  - Model: `claude-sonnet-4-20250514` (configured in `config.yaml`)
  - Purpose: Summarizes newsletters and RSS articles into digestible format; extracts structured trend data for dashboard

**Email & Communication:**
- Gmail API - Email ingestion and delivery
  - SDK/Client: `google-api-python-client`, `google-auth-oauthlib`, `google-auth-httplib2`
  - Auth: OAuth2 flow with `credentials.json` → `token.json`
  - Used in: `newsletter-digest/gmail_client.py`
  - Scopes: `gmail.readonly` (fetch emails), `gmail.send` (send digest)
  - Purpose: Fetches incoming newsletters from Gmail; sends formatted digest emails
  - OAuth callback port: `8090` (local server during auth flow)

**Content Aggregation:**
- RSS Feeds (9 configured sources)

  - Auth: None (public feeds)
  - Sources: `newsletter-digest/config.yaml`

    - Ars Technica (`https://feeds.arstechnica.com/arstechnica/index`)

    - Hacker News Best (`https://hnrss.org/best`)
    - Artificial Lawyer (`https://www.artificiallawyer.com/feed/`)

    - LawSites (`https://www.lawnext.com/feed`)
    - One Useful Thing (`https://www.oneusefulthing.org/feed`)
    - How to AI (`https://ruben.substack.com/feed`)
  - Purpose: Supplements newsletter content with curated tech/legal tech news

**Content Distribution:**
